# jms_nodejs

## 노드 버전 업데이트 하기

- 기존에 쓰던 노드 버전 : 14.16.1
- 업데이트 할 노드 버전 : 14.16.0

```
// 노드 업데이트
node -v //node 버전 확인
npm cache clean -f //캐시 삭제
sudo npm install -g n //nodejs 버전관리 플러그인
n lts //lts 버전
// npm 업데이트
npm -v //npm 버전 확인
sudo npm i -g npm //npm 업데이트(global 경로)
// 최종 버전 확인
node -v
npm -v
```
